## 🤖 Features

- **Ultra-Premium Design:** Seamless dark mode, glassmorphism, glowing hover interactions, and beautiful typography.
- **Agentic AI & Web3 Portfolio:** Detailed displays of complex RAG multi-agent pipelines and real-time LLM web platforms.
- **Immersive Animations:** Completely powered by Framer Motion leveraging spring physics, `useScroll` tracking, and highly intricate `<AnimatePresence>` modals.
- **High Performance:** Server-rendered App Router architecture providing flawless Lighthouse metrics and zero layout shift.
- **Zod Data Pipelines:** Secure client-side validated contact architecture wrapped by React Hook Form.

## 🛠 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Custom SCSS Modules
- **Animations:** Framer Motion + Canvas Confetti
- **Components:** Shadcn/UI (Radix Primitives), Lucide-React
- **State/Forms:** React Hook Form + Zod + Sonner Toasts
- **Deployment:** Vercel

## 💻 Quick Setup

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/hanzlikhan/Hanzla-portfolio-2026.git
cd Hanzla-portfolio-2026

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

Open `http://localhost:3000` with your browser to see the result.
